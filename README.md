# AWS Static Website Hosting Automation using Python SDK

## 📌 Project Overview

This project demonstrates how to automate static website hosting on AWS using Python and the boto3 SDK.  
The website files (HTML, CSS, JavaScript) are automatically uploaded to an Amazon S3 bucket, which is configured for static website hosting.

The project helps understand:
- AWS S3 static website hosting
- Python SDK automation using boto3
- IAM and AWS credential configuration
- Automated deployment of website files

---

## 🚀 Technologies Used

- Python
- boto3
- AWS S3
- AWS IAM
- HTML
- CSS
- JavaScript
